5 GerminaQuant data processing

5.1 Prepare you field book

For using the GerminaQuant is necessary that you have a data with germination values. You can use a following data as an example data. If you have a Google account you can clone the document and edit it online and download for your own analysis.

5.2 Field book

When you have your field book, you can go GerminaQuant for R and go “Fieldbook” tab. Figure 5.1.

Figure 5.1: Fieldbook interface for import your data

You can paste a Google spread sheet URL or upload a local file in xlsx format. In “Seeds (col name)” you have to write the name of the column containing the information of the number of seed sown in each experimental unit, for “Evaluations (prefix)” you have to put the prefix of the names for the evaluated days from the germination time lapse.

5.3 Germination

If the parameter in the “Fieldbook” tab are correct, in “Germination” tab will be performed and the values of the germination indices for each experimental unit will be shown. Table 2.1. GerminaQuant allows to copy or downloading the file in “csv” or “xlsx” format with the calculation of the germination variables. Figure 5.2

Figure 5.2: Dowload option for the calculated variables

5.4 Statistical

The application can perform a factorial analysis according you experimental design (crd, crbd or lsd), and calculate the analysis of variance and the mean differences through Student Newman Keuls, Tukey or Duncan test according the chosen variable.

Figure 5.3: Statitical analysis with ANOVA and mean comparison test

5.5 Graphics

Automatically after performed the statistical analysis the application will generate the graphs for the variable chosen with the mean comparison test. The interface allows customized bar or line plot and export in “tiff” format for publication quality.

Figure 5.4: Customized interface for bar or line plot

5.6 InTime

This Tab allows to visualize the germination process according the experimental factors. The interface allows customized the graphic. Figure 5.5

Figure 5.5: Germination in time plot

The application allows to plot two type of graphics, the first is the germination in percentage in time lapse and the second the relative germination that calculate the germination according the total number of germinated seeds.
